## Overview

Server-side resource that captures a player's screen and uploads it to a Discord webhook. Used for admin monitoring and anti-cheat evidence. Pulsar Framework fork of [discord-screenshot](https://github.com/jaimeadf/discord-screenshot) by Jaime Filho.

## Dependencies

- [screencapture](https://github.com/PulsarFW/screencapture)
