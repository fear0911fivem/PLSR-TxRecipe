## Overview

Damage and injury system for Pulsar Framework. Handles player health, limb damage, death, hospitalisation, prison sentencing, and fire department interactions. Integrates with the character system for persistent injury and sentence state.

---

## Dependencies

- `pulsar-core` — framework core
- `pulsar-characters` — character state persistence
- `ox_inventory` — medical item handling
- `oxmysql` — injury and sentence storage
