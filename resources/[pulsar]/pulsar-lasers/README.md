## Overview

Laser tripwire security system for Pulsar Framework. Players can place and configure laser beams that trigger alerts or actions when crossed, useful for property security and heist mechanics.

---

## Dependencies

- `pulsar-core` — framework core
- `ox_inventory` — laser item handling
