## Overview

Police speed radar gun with front/rear detection, plate locking, and remote control menu. Includes helicopter camera with rappel support. Configurable ray radius and angle offsets via `config.lua`.

## Dependencies

- [pulsar-core](https://github.com/PulsarFW/pulsar-core)
