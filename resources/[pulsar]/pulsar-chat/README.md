## Overview

Full chat system for Pulsar Framework. Supports proximity chat, global, staff, and radio channels. Sanitises input, handles formatting, and integrates with the character system for player name display.

---

## UI Development

The NUI is a React 18 + Webpack app.

```bash
cd ui
npm install --legacy-peer-deps
npm run start    # dev server with hot reload
npm run build    # production build → ui/dist/
```

---

## Dependencies

- `pulsar-core` — framework core
- `pulsar-characters` — character name for chat display
