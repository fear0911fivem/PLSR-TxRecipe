## Overview

Mobile Data Terminal for Pulsar Framework. Provides police and emergency services with a full-featured MDT — incident reports, arrest records, warrant management, BOLO system, unit dispatch, and citizen/vehicle lookups.

---

## Dependencies

- `pulsar-core` — framework core
- `oxmysql` — database layer
- `ox_inventory` — evidence and item integration
