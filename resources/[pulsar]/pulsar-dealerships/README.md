## Overview

Full dealership system for Pulsar Framework. Covers player-facing vehicle browsing, test drives, purchases, employee sales flows, showroom management, stock control, buybacks, credit checks, government fleet shops, donor dealerships, and rentals.

---

## Dependencies

- `pulsar-core` — framework core
- `pulsar-characters` — character job and finance data
- `pulsar-vehicles` — vehicle ownership and storage
- `ox_inventory` — payment handling
- `oxmysql` — stock and sales persistence
