## Overview

Vehicle customisation system for Pulsar Framework. Allows players to modify their vehicles at configurable locations — engine, brakes, transmission, paint, wheels, liveries, and saved presets.

---

## Dependencies

- `pulsar-core` — framework core
- `pulsar-vehicles` — vehicle ownership checks
- `ox_lib` — menus, notify
- `ox_inventory` — payment handling
