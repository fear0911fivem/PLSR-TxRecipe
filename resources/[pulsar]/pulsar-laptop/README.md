## Overview

In-world laptop system for Pulsar Framework. Players can open a laptop item to access apps including business management, banking, darkweb listings, and other configurable modules via a React UI.

---

## Dependencies

- `pulsar-core` — framework core
- `ox_inventory` — laptop item handling
