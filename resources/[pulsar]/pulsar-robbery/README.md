## Overview

Configurable robbery system supporting multiple locations including Fleeca branches, Maze Bank, Paleto Bay, Bobcat Security, and LS Mutual. Each location has independent configuration, cooldowns, and loot tables.

## Dependencies

- [pulsar-core](https://github.com/PulsarFW/pulsar-core)
