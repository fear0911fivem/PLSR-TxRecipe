## Overview

Full casino system for Pulsar Framework. Includes blackjack, roulette, slot machines, and the lucky wheel — all with chip-based economy integration and configurable odds.

---

## Dependencies

- `pulsar-core` — framework core
- `pulsar-characters` — player cash and chip balance
- `ox_inventory` — casino chip items
- `ox_lib` — prompts, notify
