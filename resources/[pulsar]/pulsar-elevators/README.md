## Overview

Elevator system for Pulsar Framework. Allows players to navigate multi-floor interiors via configurable elevator points. Ships with pre-configured elevators for the casino, Mt Zonah hospital, and other key locations.

---

## Dependencies

- `pulsar-core` — framework core
- `ox_lib` — menu, notify
