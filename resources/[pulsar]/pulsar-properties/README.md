## Overview

Handles player-owned properties with interior IPL management, furniture placement, and persistent ownership storage. Supports multiple interior types defined in `properties.json` with per-property configuration.

## Dependencies

- [pulsar-core](https://github.com/PulsarFW/pulsar-core)
- [oxmysql](https://github.com/overextended/oxmysql)
