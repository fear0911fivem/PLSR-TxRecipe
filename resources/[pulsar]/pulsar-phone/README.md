## Overview

Smartphone system for Pulsar Framework. Full-featured in-game phone with calls, SMS, contacts, social media, banking, job apps, darkweb, camera, and a rich app ecosystem built on React.

---

## Dependencies

- `pulsar-core` — framework core
- `oxmysql` — database layer
- `ox_inventory` — phone item handling
