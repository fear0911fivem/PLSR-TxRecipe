## Overview

Zone detection system for Pulsar Framework. Provides polygon, box, circle, and entity-based zones with enter/exit callbacks — used across the framework for area triggers, job zones, restricted areas, and interactive regions.

---

## Dependencies

- `pulsar-core` — framework core
