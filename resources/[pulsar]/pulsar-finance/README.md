## Overview

Banking and financial system for Pulsar Framework. Covers player bank accounts, cash handling, deposits, withdrawals, transfers between players, and full transaction history.

---

## Dependencies

- `pulsar-core` — framework core
- `oxmysql` — database layer
- `ox_inventory` — cash and card item handling
