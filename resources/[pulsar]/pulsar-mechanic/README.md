## Overview

Mechanic job system for Pulsar Framework. Covers vehicle repair with item requirements, tyre changes, engine work, and mechanic job integration with duty, callouts, and player-to-player repair requests.

---

## Dependencies

- `pulsar-core` — framework core
- `pulsar-jobs` — mechanic job definition
- `ox_inventory` — repair item handling
