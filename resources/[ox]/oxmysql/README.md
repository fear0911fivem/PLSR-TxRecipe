## Overview

High-performance async MySQL resource for FiveM with a promise-based API, query profiling, and a built-in UI for monitoring slow queries. Pulsar Framework fork of [oxmysql](https://github.com/overextended/oxmysql) by Overextended.

## Dependencies

- None — must be started before all other resources
